== Installation ==

This section describes how to install the plugin and get it working.

1. Unzip `woocommerce-display-cart-saleprice-x.x.zip`
1. Upload the `woocommerce-display-cart-saleprice` directory (not its contents, the whole directory) to `/wp-content/plugins/`
1. Activate the plugin through the `Plugins` menu in WordPress
